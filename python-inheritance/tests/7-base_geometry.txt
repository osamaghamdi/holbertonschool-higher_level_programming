integer_validator Method
=======================

The ``integer_validator(name, value)`` method checks if ``value`` is a positive integer.

If ``value`` is not an integer, it raises a TypeError:

::

    >>> bg.integer_validator("width", "3")
    Traceback (most recent call last):
    TypeError: width must be an integer

If ``value`` is less than or equal to 0, it raises a ValueError:

::

    >>> bg.integer_validator("width", 0)
    Traceback (most recent call last):
    ValueError: width must be greater than 0

    >>> bg.integer_validator("width", -5)
    Traceback (most recent call last):
    ValueError: width must be greater than 0

If ``value`` is a positive integer, nothing happens:

::

    >>> bg.integer_validator("width", 5)
